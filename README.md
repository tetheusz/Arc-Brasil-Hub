## 📌 O que é ARC

ARC é uma blockchain Layer 1 nativa de stablecoins, projetada como base para stablecoins, ativos tokenizados, contratos econômicos e mercados on-chain.

A rede é compatível com EVM (Ethereum Virtual Machine) e foi construída especificamente para finanças baseadas em stablecoins, utilizando USDC como gas nativo, finalidade de liquidação determinística, privacidade opcional e uma arquitetura de taxas estáveis.

O projeto é desenvolvido pela **Circle** — a empresa por trás do USDC — e se posiciona como infraestrutura fundamental para pagamentos globais, câmbio (FX) e mercados de capitais on-chain.

### Principais características

- **USDC como gas nativo** — taxas previsíveis e estáveis, sem volatilidade
- **Finalidade sub-segundo** — a rede registra finalidade em torno de 780 milissegundos para 100 validadores com blocos de 1MB
- **Privacidade opt-in** — empresas podem ativar transferências confidenciais conforme necessidade
- **StableFX** — motor de câmbio institucional integrado que permite negociação de pares de stablecoins 24/7 com liquidação on-chain
- **Compatibilidade EVM** — contratos Ethereum funcionam nativamente na rede

## 📝 Notas e resumos

### Resumo técnico (versão simplificada)

ARC resolve três problemas clássicos das blockchains tradicionais:

1. **Taxas imprevisíveis** → usando USDC como gas, os custos são fixos e previsíveis
2. **Liquidação lenta** → finalidade em menos de 1 segundo
3. **Falta de privacidade para empresas** → privacidade opcional e compatível com compliance
